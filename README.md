# Restaurant Automation System — Frontend

Frontend application for a restaurant management system built with React and Vite.<br>
[Backend Repository](https://github.com/realasdfg/restaurant)

## Features

* Authentication UI
* Menu browsing for clients
* Orders management
* Tables management
* Administration's page
* Statistics page
* Real-time updates
* Responsive design
* Integration with FastAPI backend

## Tech Stack

* React
* Vite
* JavaScript
* Axios
* React Router
* Ant Design
* Tailwind CSS

## Getting Started

### Clone the repository

```bash
git clone https://github.com/realasdfg/restaurant_frontend.git
cd restaurant_frontend
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

Application will be available at:

```text
http://localhost:5173
```
